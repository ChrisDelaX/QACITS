Welcome to QACITS’s documentation!¶

As-built documentation of QACITS modules, utilities, and notebooks. This documentation is written using Sphinx.

Credits¶

This QACITS package was written by Christian Delacroix, with contributions from Olivier Absil, Elsa Huby, and Anne-Lise Maire.

The QACITS method is an original idea from Elsa Huby.

For further information, please see the following papers:

  • Huby et al. 2015, Post-coronagraphic tip-tilt sensing for vortex phase masks: The QACITS technique

  • Huby et al. 2017, On-sky performance of the QACITS pointing control technique with the Keck/NIRC2 vortex coronagraph

  • Maire et al. 2020, Design, pointing control, and on-sky performance of the mid-infrared vortex coronagraph for the VLT/NEAR experiment
